# 智读笔记 - 智能阅读笔记管理系统

## 项目简介
智读笔记是一个智能化的阅读笔记分类和管理工具，采用苹果设计风格，提供优雅的用户体验和高效的笔记整理功能。

## 功能特性

### 🎯 智能分类
- 自动识别并分类笔记内容
- 支持四种分类标签：
  - **#思考** - 有启发的句子，并附上思考
  - **#知识** - 有趣的新知识，可以是概念或新认知
  - **#行动** - 指导行动的内容，并附上场景
  - **#金句** - 经典的句子，可附上思考
  - **仅划线** - 纯划线内容

### 🎨 现代化界面
- 苹果设计风格，毛玻璃效果
- 响应式设计，适配各种设备
- 流畅的动画和交互效果
- 快捷复制功能

### 🔧 智能解析
- 自动合并个人想法和原文内容
- 智能跳过章节标题
- 内容去重处理
- 支持多种笔记格式

## 使用方法

### 1. 输入笔记内容
在文本框中粘贴您的读书笔记，格式可以包含：
- 分类标签（#思考、#知识、#行动、#金句）
- 原文内容（以"原文："开头）
- 划线内容（以"◆"开头）
- 个人想法内容

### 2. 点击分类
点击"分类笔记"按钮，系统会自动：
- 识别分类标签
- 合并相关笔记内容
- 跳过无关章节标题
- 去除重复内容

### 3. 查看结果
通过顶部标签切换查看不同分类的笔记：
- **全部** - 显示所有分类的笔记
- **思考/知识/行动/金句** - 按分类查看
- **仅划线** - 纯划线内容
- **未分类** - 未标记分类的内容

### 4. 快捷操作
- 点击笔记卡片任意位置复制内容
- 使用右上角复制按钮
- 清空功能重新开始

## 使用建议

### 最佳实践
1. **阅读前**：思考为什么想要读这本书
2. **阅读时**：针对不同的划线打上相应的标签
3. **阅读后**：导出划线内容，使用本工具整理

### 输入格式示例
```
#思考
这是我的个人思考内容
原文：这是书中的原文内容

#知识
◆ 这是纯划线内容
```
