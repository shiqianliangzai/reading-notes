# 智读笔记

一个用于整理阅读笔记的轻量前端工具，解析规则以项目内的 [笔记处理规则文档](D:/MyWarehouse/reading-notes/%E7%AC%94%E8%AE%B0%E5%A4%84%E7%90%86%E8%A7%84%E5%88%99%E6%96%87%E6%A1%A3.md) 为准。

## 现在的简化理解

可以把输入规则理解成 4 种行：

- `#思考 / #知识 / #行动 / #金句`：开启一个分类块
- `◆ 日期发表想法`：开始收集个人想法，直到遇到 `原文：`
- `原文：`：补到当前块里
- `◆ 普通划线`：没有原文时并入当前分类块；有原文且不重复时，拆成独立“仅划线”

另外还有两条补充规则：

- 连续两个空行后，第一个非标记行会被当作章节标题跳过
- 笔记块首尾空白会去掉，中间空白保留

## 代码结构优化

- 解析器现在按状态拆成多个小处理函数，而不是把所有分支都塞在一个大循环里
- UI 结果区改成更清晰的卡片样式，增加摘要感和层级感
- 仍然不显示“未分类”分类

## 文件说明

- `index.html`：页面结构
- `styles.css`：界面样式
- `script.js`：解析与渲染逻辑
- `笔记处理规则文档.md`：规则说明
- `test_example.txt`：测试示例

## 使用方式

1. 打开 `index.html`
2. 粘贴阅读笔记
3. 点击“整理笔记`
4. 按分类查看结果
5. 点击卡片右上角“复制”复制单条内容
